## 5. User Data Processing
- As a user of any Discord server where we provide the Service, we process the following data:
  * [Click to review the user data used](https://discord.com/developers/docs/resources/user#user-object).
- This data enables searching and displaying information about Discord server users and performing commands quickly. Therefore, if you are on a server where we provide the Service and wish to remove your data from temporary storage, you should leave that server.
- The legal basis for processing this data is Article 6(1)(f) GDPR, which is the legitimate interest of the Service Provider in ensuring the basic functionality of the Service. Without this, the Service would not operate and its functionalities would be significantly limited.
